# 🎮 College Escape – 3D Unity Exploration Game

College Escape is my first Unity 3D project, designed as an exploration and escape-based game set in a virtual recreation of my college campus.In this game, the player controls a character who explores a detailed 3D model of the college while completing various tasks and challenges. The objective is to collect items such as keys, clues, and perks that help unlock restricted areas and ultimately escape the campus. 
<br>
The gameplay is inspired by exploration-survival mechanics similar to Granny, where strategy and stealth play an important role. An enemy character roams the environment, creating obstacles and danger. Players must avoid detection, hide when necessary, and carefully plan their actions to complete tasks successfully.
College Escape is my first Unity 3D project, featuring an exploration and escape-style gameplay set in a virtual recreation of my college campus. The game focuses on immersive exploration, task-based progression, and stealth mechanics.
<br>
Players control a character who navigates through the college environment, interacting with different areas and completing objectives. Throughout the game, the player must collect important items such as keys, clues, and perks that help unlock restricted sections of the map. These elements are essential for progressing and eventually escaping the campus. 
<br>
The gameplay is inspired by survival-exploration games, where strategy and careful movement are important. An enemy character actively roams the environment, creating obstacles and adding tension. The player must avoid being caught by hiding, planning movements, and completing tasks efficiently. The project demonstrates core Unity concepts such as 3D environment design, character control, basic AI behavior, and interactive gameplay systems. It serves as a foundational project showcasing my learning and understanding of game development using Unity and C#. Overall, the game combines exploration, puzzle-solving, and stealth elements to create a simple but engaging experience centered around escaping from a realistic college setting.
<br>
### 🧩 Features
- 🏫 Realistic 3D representation of my college campus  
- 🎯 Task-based progression system (keys, clues, perks)  
- 🔐 Locked areas that require exploration and problem-solving  
- 👤 Third-person character exploration  
- 👹 Enemy AI that adds challenge and tension  
- 🕵️ Stealth mechanics (hiding and avoiding danger)  

---

### 🎯 Objective
Explore the environment, complete all required tasks, unlock the main gate, and escape while avoiding the enemy.

---

### 🚀 Tech Stack
- Unity (Game Engine)  
- C# (Scripting)
