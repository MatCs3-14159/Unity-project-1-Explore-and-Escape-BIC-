# 🎮 College Escape – 3D Unity Exploration Game

**College Escape** is my first Unity 3D project, designed as an exploration and escape-based game set in a virtual recreation of my college campus.

## 🧩 Overview
In this game, players control a character who explores a detailed 3D environment while completing tasks and challenges. The goal is to collect essential items like **keys, clues, and perks** to unlock restricted areas and ultimately escape the campus.

The gameplay focuses on **exploration, puzzle-solving, and stealth mechanics**, creating an engaging and strategic experience.

## 🎯 Gameplay
- Explore the college environment freely  
- Complete tasks to progress through the game  
- Collect keys, clues, and perks  
- Unlock restricted areas  
- Avoid an enemy character that roams the map  
- Use stealth mechanics like hiding and careful movement  

## 👹 Challenge Element
An enemy AI continuously patrols the environment, adding tension and difficulty. Players must avoid detection and plan their actions strategically to survive and escape.

## 🧪 Features
- 🏫 Realistic 3D college campus  
- 🎯 Task-based progression system  
- 🔐 Locked areas with puzzles  
- 👤 Third-person character control  
- 👹 Enemy AI behavior  
- 🕵️ Stealth gameplay mechanics  

## 🎯 Objective
Explore the environment, complete all tasks, unlock the main gate, and escape without being caught.

## 🚀 Tech Stack
- **Unity** (Game Engine)  
- **C#** (Scripting)  

## 📌 Project Purpose
This project demonstrates my understanding of:
- 3D environment design  
- Character movement and control  
- Basic AI systems  
- Interactive gameplay mechanics  

It serves as a foundational project showcasing my learning journey in game development using Unity and C#.
